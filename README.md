Blinkit Sales Analysis using Python & Power BI
📌 Project Overview
This project analyzes Blinkit grocery sales data using Python and Power BI.
The objective of this project is to explore sales patterns, customer behavior, and product performance using data analytics techniques.
The analysis includes data cleaning, exploration, visualization, and business insights that help understand how Blinkit products perform across different outlets and categories.
This project is useful for data analytics beginners and freshers to demonstrate practical skills in Python, data analysis, and dashboard creation.

Key Insights:-
Some important insights discovered:
Certain product categories generate higher revenue.
Medium sized outlets often perform better in sales.
Low fat products show strong customer demand.
Sales vary significantly across different outlet locations.
